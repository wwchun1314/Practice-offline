# Practice-offline
适用于PC电脑、手机等设备的离线刷题网页，因所有题目及答案都保存在网页中，有作弊的风险，所以不适用于各类考试。



## 使用说明：

1、先将题目按模板进行保存（txt或xlsx版本都可，xls未适配）。
2、运行python文件（将题库数据保存到网页中）
  1、选中题库文件，2、选中网页模板文件，3、输入要生成的网页文件名称，4、点击开始生成。
3、手机或电脑使用浏览器打开。



### Python文件（题库生成器.py）的功能：

1、将题库模板（或自己题库）改动为Json格式的数据

2、将Json数据插入到网页中

3、图形化界面



text2Json2Html.py为旧版控制台程序，无窗口界面
